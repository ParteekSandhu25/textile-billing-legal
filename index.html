Textile Billing Software

Textile Billing Software is a desktop application developed to help textile businesses manage billing operations and securely back up billing data.

Application Overview

The software operates locally on the user's computer. All billing data is stored on the user’s device. The application offers optional integration with Google Drive solely for the purpose of backing up billing files created by the software.

Google Drive Integration

Textile Billing Software uses the Google Drive API with the scope:
https://www.googleapis.com/auth/drive.file

This permission allows the application to create and manage backup files that are generated by the software. The application does not access, read, or modify files that were not created by the application itself.
