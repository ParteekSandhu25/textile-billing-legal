Textile Billing Software

Textile Billing Software is a desktop application built using Electron. It helps textile businesses manage billing and securely back up billing data to the user's Google Drive account.

Features

  • Billing management
  • Invoice generation
  • Automatic Google Drive backup

The application only accesses Google Drive to store backup files created by the software. It does not access, read, or modify other user files.
