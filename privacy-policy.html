Privacy Policy

Effective Date: January 1, 2026

1. Overview

Textile Billing Software is a desktop-based application. We are committed to protecting user privacy and ensuring transparency regarding data usage.

2. Data Collection

The application does not collect, transmit, or store personal data on any external servers. All billing data is stored locally on the user’s computer.

3. Google Drive API Usage

The application requests access to Google Drive using the scope:
https://www.googleapis.com/auth/drive.file

This permission is used exclusively to:

  • Create backup files generated by the application
  • Update backup files created by the application
  • Access only files created by the application

The application does not access, read, modify, or delete any other files in the user’s Google Drive.

4. Data Storage and Security

Backup files are stored directly within the user’s own Google Drive account. The developer does not have access to user Google accounts or stored files.

OAuth access tokens are stored locally on the user’s device and are not transmitted to any external server controlled by the developer.

5. Data Sharing

We do not sell, rent, share, or transfer user data to third parties.

6. User Control and Revocation

Users may revoke Google Drive access at any time by visiting:
https://myaccount.google.com/permissions

7. Changes to This Policy

This Privacy Policy may be updated periodically. Continued use of the application constitutes acceptance of the updated policy.
