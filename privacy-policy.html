Privacy Policy

Effective Date: January 1, 2026

1. Information We Collect

Textile Billing Software does not collect or store personal user data on external servers. All billing data is stored locally on the user's device.

2. Google Drive Access

The application requests access to the user's Google Drive using the scope: https://www.googleapis.com/auth/drive.file

This access is used only to create and manage backup files generated by the application. The software does NOT read, modify, or delete files that were not created by the application.

3. Data Storage

Backup files are stored directly in the user's own Google Drive account. We do not have access to user Google accounts outside the authorized scope.

4. Data Sharing

We do not sell, share, or transfer user data to third parties.

5. Revoking Access

Users can revoke Google Drive access at any time from: https://myaccount.google.com/permissions
